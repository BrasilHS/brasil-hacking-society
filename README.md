# brasil-hacking-society
Brasil Hacking Society é uma iniciativa colaborativa voltada à pesquisa e desenvolvimento em segurança da informação, engenharia de software e infraestrutura.

O projeto tem como objetivo unir pesquisadores, desenvolvedores e entusiastas da área para promover aprendizado técnico, compartilhamento de conhecimento e construção de ferramentas open source voltadas à segurança ofensiva e defensiva.

Todas as atividades são conduzidas de forma ética, responsável e em conformidade com a legislação vigente.
